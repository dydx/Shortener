Shortener -- simple URL shortening with Sinatra and OpenKeyVal.org

1.) Basic UI:

 ![Homepage](http://imgur.com/af7TS.png)
 ![Homepage with URL in form](http://imgur.com/xQV0J.png)
 ![Submitted url with response](http://imgur.com/WM3kZ.png)

2.) Underlying technologies:

  - Sinatra (http://www.sinatrarb.com/)-
  - OpenKeyval (http://openkeyval.org/)
  - FlyingV (https://github.com/tomash/flyingv)

3.) Configuration is pretty simple and the settings for such are located in the file shortener.rb.
